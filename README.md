## Sistema Cafetería (JavaFX)

Proyecto de escritorio con JavaFX y PostgreSQL para login de usuarios y control de roles (ADMIN, CAJERO, BARISTA).

## Funciones
Login de usuarios
Validación de contraseña y rol
Dashboard según usuario
Conexión a base de datos
